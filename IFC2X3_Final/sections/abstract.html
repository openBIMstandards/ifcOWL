Abstract

This ifcOWL ontology is automatically generated from the EXPRESS schema 'IFC2X3_Final' using the 'IFC-to-RDF' converter developed by Pieter Pauwels (pipauwel.pauwels@ugent.be), based on the earlier versions from Jyrki Oraskari (jyrki.oraskari@aalto.fi) and Davy Van Deursen (davy.vandeursen@ugent.be). The ontology is identical to the ontology that is generated from the EXPRESS schema 'IFC2X3_Final.exp' using the 'genOnto' converter developed by Walter Terkaj (ITIA-CNR, walter.terkaj@itia.cnr.it).
